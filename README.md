Sistema de Inventario con POO

Sistema de gestión de inventario construido en Python puro
usando Programación Orientada a Objetos.

Problema que resuelve
Gestionar productos, stock y alertas de inventario bajo
de forma automática desde un archivo CSV.

Funcionalidades
- Agregar productos con validación de datos
- Actualizar stock (entradas y salidas)
- Buscar productos por nombre o categoría
- Alertas automáticas de stock bajo o agotado
- Reporte completo con valor total del inventario

Tecnologías
- Python 3.10+
- POO: clases, herencia, encapsulación
- Módulos: `csv`, `os`, `datetime`
  
Estructura
inventario-python-poo/
│
├── inventario.ipynb   # Notebook principal

├── inventario.csv     # Dataset de ejemplo

└── README.md


Cómo usarlo
1. Abre el notebook en Google Colab
2. Ejecuta todas las celdas en orden
3. Usa los métodos para gestionar tu inventario

Lo que aprendí
- Diseño de clases con responsabilidad única
- Diccionarios como estructura de datos principal
- List comprehensions para filtrar datos
- Persistencia de datos con CSV
- Validación robusta con excepciones personalizadas
